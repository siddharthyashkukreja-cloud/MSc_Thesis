\section{Agent-Layer Parameters}\label{app:params}
This appendix lists the agent (market-layer) parameters of the model. Table~\ref{tab:app-agent} gives the
standard model, calibrated separately for the calm and stressed regimes by surrogate-assisted simulated
moments;  Table~\ref{tab:app-synth}
gives the synthetic extension, in which five agent parameters are re-calibrated \emph{jointly} with the four
parameters of the two-factor stochastic-volatility fundamental. The \texttt{Code} column gives the matching
identifier in the released model (\texttt{model/globals.py}). The clearing tier --- ten banking and five
non-banking members (five carrying client books) and ninety clients --- use risk parameters set by design,
not reproduced here.
